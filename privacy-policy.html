Privacy Policy

Effective Date: July 29, 2025

This Privacy Policy describes how Modern Industries (“we”, “us”, or “our”) collects, uses, and protects information when you use the Issue Tracking App (the “App”), developed exclusively for internal operational use by Modern Industries employees.

1. Information We Collect

The App does not collect personally identifiable information (PII) such as:

  • Full name
  • Email address
  • Phone number
  • Precise or approximate location
  • Contacts or calendar events

The App collects limited technical data solely for performance diagnostics and compatibility:

  • Device type and model (e.g., Android)
  • Operating system version
  • Application version

This information is never shared or sold to any third party and is used only to improve the performance and reliability of the App.

2. Image Uploads and Cloud Storage

During configuration activities (such as adding machines), authorized personnel (e.g., administrators) may upload images using the App. These images are securely stored using Cloudinary, a GDPR- and CCPA-compliant cloud-based media storage service.

Cloudinary may store data on servers located outside India, including the United States. All transfers are encrypted in transit using HTTPS, and the service provider maintains compliance with international data protection frameworks.

Image URLs are stored in a secure, access-controlled cloud database. This database is hosted in a private cloud infrastructure with strict access and encryption protocols. Data is never made public or shared with any third parties without explicit authorization from Modern Industries.

3. Permissions

The App requests the following permissions at runtime, only for use by authorized administrative personnel while performing specific tasks such as machine profile configuration:

  • Camera: To capture machine images during setup.
  • Storage: To select and upload existing machine-related images from the device.

These permissions are not requested or required during regular usage by standard employees. The App does not access these resources unless explicitly initiated by authorized users. Permission access is governed by role-based access control mechanisms implemented within the App’s authentication system.

4. Third-Party Services

The App does not include advertising, analytics, or tracking SDKs. The only third-party service integrated is Cloudinary, which is used strictly for secure, internal media hosting as part of configuration workflows.

5. Children’s Privacy

This App is intended strictly for use by authorized employees of Modern Industries. It is not directed toward children under the age of 13. We do not knowingly collect, process, or retain data from minors. If such data is identified, it is immediately and permanently deleted.

6. Data Security

All data transmissions are encrypted using HTTPS. The backend, including cloud-based infrastructure used for image metadata and media storage, is protected by modern security protocols including encryption at rest and in transit.

7. Data Retention and Deletion

Uploaded images and associated metadata are retained only as long as necessary to complete configuration or issue resolution tasks. Data is periodically reviewed and deleted based on internal retention schedules. Specific deletion requests can be submitted via an employee’s reporting manager or IT administrator.

8. Policy Updates

This Privacy Policy may be updated from time to time due to operational, legal, or technical changes. Updates will be posted within the App and/or on our official website, with the revised effective date clearly indicated. Continued use of the App constitutes acknowledgment of the updated policy.
